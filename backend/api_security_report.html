1. Security Architecture Summary

Authentication Mechanisms

  • Multi-Auth Support: Supports JWT, Admin Cookies, and API Keys.
  • Local Verification: Zero-network JWT validation via private secrets.
  • Device Binding: Session binding to IP and User-Agent patterns.

Access Control Logic

  • RBAC Hierarchy: Specialized roles (Super, Risk, Payouts, KYC).
  • KYC Enforcement: Hardware-level guard for high-risk operations.
  • Server-Side Pricing: All payment amounts re-validated at capture.
